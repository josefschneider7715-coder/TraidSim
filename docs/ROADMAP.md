# Roadmap

## Phase 1 – MVP lokal

- App startet
- Watchlist wird analysiert
- Signale werden berechnet
- Charts werden dargestellt
- Backtest läuft
- CSV Export

## Phase 2 – bessere Backtests

- Buy-and-Hold Vergleich
- CAGR
- Sharpe Ratio
- Profit Factor
- monatliche Renditen
- Parameter-Optimierung

## Phase 3 – Speicherung

- SQLite
- Watchlists speichern
- Signalhistorie speichern
- letzte Analysen anzeigen

## Phase 4 – Serverbetrieb

- Docker
- Hetzner VPS
- Login
- HTTPS
- automatische tägliche Scans

## Phase 5 – Benachrichtigung

- E-Mail
- Telegram
- Push

## Phase 6 – Profiversion

- FastAPI
- React/Next.js
- PostgreSQL
- mehrere Datenanbieter
- Multi-Strategie-System
